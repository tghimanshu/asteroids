# Asteroids Game

This is a simple implementation of the classic Asteroids game using Python and the Pygame library. The player controls a spaceship that can rotate and thrust forward to navigate through space while avoiding and destroying asteroids.

## Getting Started

### Prerequisites
- Python 3.x
- Pygame library
- uv
